# 🚀 AI Career Operating System

A personal career dashboard built with React.js that helps students and job seekers manage their entire job search in one place.

## ✨ Features

- 📋 **Application Tracker** — Track all job applications with status
- 📅 **Interview Scheduler** — Schedule and manage interviews by date
- 💡 **Skills Tracker** — Track learning progress for each skill
- 🎯 **Daily Goals** — Set and complete daily career goals
- 📄 **Resume Score** — Check resume strength with a checklist
- 🏠 **Dashboard** — See everything at a glance with real stats

## 🛠️ Built With

- React.js
- JavaScript (ES6+)
- CSS3
- Git and GitHub

## 🚀 Getting Started

### Install dependencies
npm install

### Start the app
npm start

### Open in browser
http://localhost:3000
